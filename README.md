- Nominal Roll
- Seating Generator
- Reports
- Teacher Profile
- Account Settings

## Description

SmartSeat is a web-based Examination Seating Management System developed for Malik Deenar College of Graduate Studies. The application automates examination seating arrangements, manages students and rooms, generates seating plans, and helps students locate their examination seats quickly.
